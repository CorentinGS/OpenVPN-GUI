# OpenVPN-GUI

A simple GUI implementation for OpenVPN to easily add my configuration options.

Been tested with surfshark config files

## How to configure

Set your environment variable
```
VPN_USERNAME=[your username]
VPN_PASSWORD=[your password]
```
